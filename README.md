# DecodeLabs-Internship-3
SQL analysis


This project involves performing exploratory data analysis on an e-commerce dataset using SQL in PostgreSQL (pgAdmin). The dataset was imported from a cleaned CSV file into a database table, where SQL queries were used to explore, filter, and analyze the data. Key operations included SELECT statements, WHERE filtering and ORDER BY sorting.

The goal of the project was to extract meaningful business insights such as top-selling products, revenue distribution across categories, and customer purchasing patterns. Data cleaning steps were also performed, including handling text-based numeric values and converting them into appropriate data types for analysis.
